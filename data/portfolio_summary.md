## 핵심 프로젝트 상세

### Project 1: Doeat Merchant Squad - 유저 이탈률 30% 감소
- **배경**: 유저 이탈률 급증 문제
- **분석**: 3가지 핵심 결함 발견
  1. 배달 지연: 예상 조리 완료 시간 대비 25분 초과
  2. 주문 취소: 팀 주문 대기 10분 후 취소
  3. 매장 해지: 이전 주문 매장 해지
- **문제 정의**: "높은 매장 해지율이 유저 이탈 증가의 주원인"
- **성과**:
  - 매장 부활: 250개 해지 매장 중 180개 부활
  - 해지 예방: 주간 해지율 15% → 1%
  - 유저 이탈률: 30% 감소
- **실패와 교훈**: 내가 떠난 후 해지율 1% → 5% 상승. 마이크로매니징 문제 인식. "나 없이도 돌아가는 시스템을 만드는 것이 PO의 역량"
- **핵심 에피소드**: "손을 더럽히는 팀 / 문제의 본질" - 250개 매장 직접 전화, 100개 매장 부활

### Project 2: Doeat Rider Squad - 3PL 시스템 구축
- **배경**: 프리랜서 라이더 의존 → 공급 불안정 → 배달 지연
- **문제 정의**: "불안정한 라이더 공급이 배달 지연의 주원인"
- **접근**: 지역 에이전시(지점) 파트너십으로 정규직 라이더 확보
  - 100명 개별 프리랜서 관리 → 10개 에이전시 x 10명 라이더
- **성과**:
  - 배달 지연율: 5% → 1%
  - 유닛 이코노믹스: -430원 → +10원
  - 운영비: 30원 → 10원

### Project 3: MoneyWalk Japan - Revenue per DAU 2배, LTV/CAC 2.5 달성
- **배경**: DAU당 유닛 이코노믹스 -17원 (수익 51원, 비용 69원)
- **문제 정의**: "Revenue per DAU 2배가 LTV/CAC 개선의 핵심"
- **접근**: In-app 광고 + Offerwall & affiliate 두 채널 최적화
- **In-app 광고 최적화 (+15원)**:
  - 판다 키우기 피처: D1 60%, D3 55% 리텐션
  - 룰렛형 광고: 광고 피로도 최소화 (+2.5원)
  - 1:1 배틀: 인게이지먼트로 자연 광고 참여 유도 (+3원)
  - 판다 + Money Farm 동시 런칭: 카니발라이제이션 10%만 (+9원)
- **Offerwall 최적화 (+25원)**:
  - Adjoe 게임 오퍼월 도입 (+5원)
  - 세그먼트별 전략: 기존 유저 디스커버리 / 신규 유저 액티베이션 (+5원)
  - 판다→Adjoe 전환 ver1: 실패 (JTBD 미스매치)
  - 판다→Adjoe 전환 ver2: 성공 (+15원) - 대나무 미션에 영구 가이드 추가
- **핵심 인사이트**: "비사용자를 사용자로 전환" - 80%가 이전에 방문만 하고 이탈했던 유저

### Project 4: MoneyWalk Discovery - D7 리텐션 +10%
- **배경**: 지속 가능한 비즈니스 모델 구축 후, "유저의 건강 습관 행동 변화" 도전
- **문제 정의**: "DAU 중 5,000보 이상 걷는 비율" 핵심 지표
  - Ability(능력) 개선 vs Motivation(동기) 개선
  - 데이터 분석: 유저의 걷기 능력 자체는 변하지 않음 → 동기 부여에 집중
- **Streak 시스템**:
  - 3-variant A/B 테스트
  - 3일 연속 달성률: 20% → 25% → 28%
  - 인사이트: Day 4-5가 습관 형성 변곡점
- **Streak Freeze 기능**:
  - D3+ 리텐션 3-5%p 개선
  - 3일+ 스트릭 달성: 20% → 26%
  - D7+ 리텐션 10% 개선
